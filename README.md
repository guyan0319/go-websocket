# go-websocket
基于golang实现的分布式聊天系统，支持一对一聊天，聊天室等功能。为了测试方便暂时没有将数据存入数据库，后期会加入数据库，也可自行加入数据库，方便永久存储聊天内容，以及支持消息必达等功能。



依赖包

```
github.com/go-redis/redis
github.com/gin-gonic/gin
github.com/gorilla/websocket
```
